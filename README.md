# Усманка-уикенд 20-21 июня 2026

Профессиональный лендинг для активных туров выходного дня на природе Воронежской области.

## О проекте

Сайт предлагает 6 тематических туров на Усманку с отправлением из Воронежа:

1. **Танцевальный девичник** (от 6 000₽) - хореография, видеосъёмки у замка, йога, винная дегустация
2. **Английская Усманка** (от 6 000₽) - языковое погружение, игры и квесты на английском
3. **Кулинарная Усманка** (от 8 000₽) - мастер-классы от шеф-повара, винная дегустация
4. **Фитнес Усманка** (от 8 000₽) - йога, тренировки, сплавы, консультации по питанию
5. **Фитнес-свидания** (от 10 000₽) - знакомства через спорт, парные тренировки
6. **Гитара-Усманка** (от 5 000₽) - обучение игре на гитаре с нуля за выходные

## Установка и запуск

### 1. Установите зависимости

```bash
pip install -r requirements.txt
```

### 2. Запустите сервер

```bash
python main.py
```

Сервер запустится на http://localhost:8000

## Структура проекта

- `main.py` - основной файл приложения FastAPI
- `source/data.py` - данные всех туров
- `templates/` - HTML шаблоны страниц
  - `index.html` - главная страница со списком туров
  - `tour.html` - динамический шаблон для страниц туров
  - `dance.html`, `english.html`, `culinary.html`, `fitness.html`, `fitness-dating.html`, `guitar.html` - статические страницы туров
- `static/` - статические файлы (CSS, JS, изображения)
  - `js/common.js` - общий JavaScript для всех страниц
  - `css/style.css` - дополнительные стили

## API Endpoints

- `GET /` - главная страница
- `GET /tour/{tour_key}` - страница конкретного тура (dance, english, culinary, fitness, fitness-dating, guitar)
- `POST /api/booking` - обработка бронирования
- `GET /health` - проверка работоспособности

## Настройка оплаты

Для настройки интеграции с ЮKassa:

1. Зарегистрируйтесь на https://yookassa.ru/
2. Получите Shop ID и секретный ключ
3. Обновите конфигурацию в `static/js/common.js`
4. Настройте backend API для создания платежей

## Разработка

Проект использует:
- **FastAPI** - веб-фреймворк
- **Jinja2** - шаблонизатор
- **Tailwind CSS** - CSS фреймворк
