# Усманка-уикенд 20-21 июня 2026

Профессиональный лендинг для активных туров выходного дня на природе Воронежской области.

## О проекте

Сайт предлагает 6 тематических туров на Усманку с отправлением из Воронежа:

1. **Танцевальный уикенд** (от 8 000₽) - хореография, видеосъёмки, йога, винная дегустация
2. **Английская Усманка** (от 6 000₽) - языковое погружение, игры и квесты на английском
3. **Кулинарная Усманка** (от 8 000₽) - мастер-классы от шеф-повара, винная дегустация
4. **Фитнес Усманка** (от 8 000₽) - йога, тренировки, сплавы, консультации по питанию
5. **Фитнес-свидания** (от 10 000₽) - знакомства через спорт, парные тренировки
6. **Гитара-Усманка** (от 5 000₽) - обучение игре на гитаре с нуля за выходные

## Быстрый старт

### Локальная разработка

```bash
# Установка зависимостей
npm run install-all

# Запуск в режиме разработки
npm run dev
```

Откроется:
- React приложение: http://localhost:3000
- Node.js сервер: http://localhost:5000

### Сборка для продакшена

```bash
# Сборка React приложения
npm run build

# Деплой на GitHub Pages (если настроен)
npm run deploy:gh-pages
```

## Деплой на GitHub Pages

### Автоматический деплой

1. Загрузите код в GitHub репозиторий
2. Включите GitHub Pages в настройках репозитория:
   - Settings → Pages → Source: GitHub Actions
3. При каждом push в main/master ветку сайт будет автоматически обновляться

### Ручной деплой

```bash
cd client
npm install
npm run deploy
```

## Настройка бронирования

По умолчанию бронирование работает через mailto. Для интеграции с внешними сервисами:

1. Отредактируйте `client/src/components/BookingModal.js`
2. Замените функцию `handleSubmit` на вызов вашего API
3. Добавьте необходимые зависимости в `client/package.json`
